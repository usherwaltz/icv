iCV is a serverpod / flutter app used to create CV's and showcase the publishers CV for recruiters.

Reference PRD.md for product requirements and PLAN.md (Implementation plan) to see what step we are on and what to do next.

On any code change (only .dart files) run dart format and dart analyze.

Whenever writing code, if it's serverpod code, reference the serverpod_mcp server

Whenever writing Flutter or dart specific code, use the dart mcp servers.

If possible, all UI components should be done using ShadCN_UI.

Check .agents/shadcn_ui directory for examples.
